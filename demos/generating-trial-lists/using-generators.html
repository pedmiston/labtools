Using Generators¶

This demo exhibits some of the advantages to using generators for more complicated trial generation.

# this demo is in the root dir of the labtools project, available on GitHub
import sys, os
sys.path.insert(0, os.path.abspath('..'))

import pandas as pd
import numpy as np

from labtools.trials_functions import *
from labtools.generator_functions import *

items = ['red','orange','yellow','green','blue','indigo','violet']
items = pd.DataFrame({'item':items}, index = pd.Index(range(len(items)),
                                                      name = 'item_id'))

stroop = pd.DataFrame({'letters':items['item'].values})
stroop = extend(stroop, reps = 4, rep_ix = 'letters_iter',
                row_ix = 'letters_id')

circulargenerator¶

The basic workhorse of the generator functions is a generator that yields rows from a pandas.DataFrame circularly, i.e., once you get to the bottom, you go back to the top of the list. However, it doesn’t do much on it’s own, so it’s a private function in the labtools package.

Yield rows from source with optional randomization.

  • prng (numpy.random.RandomState or None) – Optional randomizer.
Returns:

Yields rows from source.

pandas.Series

generate¶

generate is the lowest level of trial generation. You have a trial list, and a source. generate takes from the source and adds to the trial list.

labtools.generator_functions.generate(frame, source, source_cols=None, seed=None)[source]¶

Adds columns to a trial list from a source using a circular generator.

Parameters:
  • frame (pandas.DataFrame) – Trial list.
  • source (pandas.DataFrame) – Source list.
  • source_cols (str, list, dict, or None) – Columns of source to add to frame. Defaults to adding all columns of source. If source_cols is a dict, keys will be renamed to values.
  • seed (int or None) – Seed random number generator.
Returns:

The frame with additional source_cols from source.

pandas.DataFrame

generate_but_not¶

However, often you won’t just want to randomly stick together your trial list with some source, as in the case of a Stroop experiment. We want to be able to control which letters go with which colors so that they don’t match. Building on generate is the function generate_but_not which breaks the trial list into chunks before using _circular_generators.

labtools.generator_functions.generate_but_not(frame, source, on, source_cols=None, seed=None)[source]¶

Adds columns to a trial list based on non-matching values in source.

For more information:
  • on how the columns are added, see generate_by_group().
  • on how the non-matching values are selected, see create_source_map()
Parameters:
  • frame (pandas.DataFrame) –
  • source (pandas.DataFrame) – Full options to split into the source_map.
  • on (str or list) – Column names to match source and frame on.
  • source_cols (str, list, dict, or None) – Columns of source to add to frame. Defaults to adding all columns of source. If source_cols is a dict, keys will be renamed to values.
  • seed (int or None) – Seed random number generator. If None the result will not be randomized.
Returns:

The frame with additional source_cols from sources.
Return type:

pandas.DataFrame

Generic functions¶

More complicated generation can be accomplished by using the two workhorse functions generate_by_group and create_source_map.

generate_by_group¶

labtools.generator_functions.generate_by_group(frame, by, source_map, source_cols=None, seed=None)[source]¶

Adds columns to a trial list from multiple sources.

Splits a trial list into chunks to add columns from various sources. Chunks are paired with sources based on unique values in frame[by]. See generate() for more details.

Parameters:
  • frame (pandas.DataFrame) – Trial list.
  • by (str) – Grouping column in frame. Unique values are used as keys to get sources from source_map.
  • source_map (dict) – Container of source lists. Keys are unique values of frame[by]. Values are pandas.DataFrame sources.
  • source_cols (str, list, dict, or None) – Columns of source to add to frame. Defaults to adding all columns of source. If source_cols is a dict, keys will be renamed to values.
  • seed (int or None) – Seed random number generator. If None the result will not be randomized.
Returns:

The frame with additional source_cols from source.
Return type:

pandas.DataFrame

create_source_map¶

labtools.generator_functions.create_source_map(source, on, source_keys, comparison_func)¶

Splits a source into groups using a comparison function.

Parameters:
  • source (pandas.DataFrame) – Source list.
  • on (str) – source[on] is used as an operand in comparison_func.
  • source_keys (list) – Secondary operand in comparison_func.
  • comparison_func (function) – Function to compare source[on] to source_keys. Must return a boolean mask the same length as source.
Returns:

Map of source_keys to subsets of the source that satisfy the comparison_func
Return type:

dict
